# Complexity Cheatsheet

## Common Time Complexities

| Complexity | Meaning | Example |
|---|---|---|
| O(1) | Constant time | Array index lookup |
| O(log n) | Logarithmic time | Binary search |
| O(n) | Linear time | Single loop |
| O(n log n) | Linearithmic time | Efficient sorting |
| O(n^2) | Quadratic time | Nested loops |
| O(2^n) | Exponential time | Generating subsets |
| O(n!) | Factorial time | Generating permutations |

---

## Common Space Complexities

| Complexity | Meaning |
|---|---|
| O(1) | Constant extra memory |
| O(n) | Memory grows with input size |
| O(n^2) | Matrix or pairwise storage |
